ซอฟต์แวร์ตรวจสอบความปลอดภัยของวัว Cow Strike จะทําหน้าที่ตรวจสอบวัวทุกตัวที่จะถูกรีดนม แต่หลังจากเหตุการณ์ที่ซอฟต์แวร์นี้เกิดทํางานผิดพลาด นักศึกษาจึงได้รับหน้าที่ให้เขียนโปรแกรมเพื่อตรวจสอบโปรแกรมนี้อีกต่อหนึ่ง

ความต้องการของโจทย์ (Requirements)
นักศึกษาจะต้องเขียนโปรแกรมแบบ MVC (Model-View-Controller) สําหรับการตรวจสอบน้ํานมที่ออกมาจากวัวอีกต่อหนึ่ง
1) เขียนโปรแกรมโดยใช้แนวคิด MVC (Model-View-Controller) Design Pattern หรือแนวคิดอื่น ๆ ที่คล้ายคลึง โดยใช้ภาษาโปรแกรม Java
2) วัวแต่ละตัวจะมีข้อมูล ให้นักศึกษาเริ่มจากการสร้างข้อมูลของวัวเก็บไว้ก่อน ประกอบด้วยรหัสของวัวเป็นเลข 8 หลักที่เลขตัวแรกไม่ใช่ 0, พันธุ์ของวัวตามสี (มี 2 สี คือสีขาว และสีน้ําตาล), และอายุของวัว เก็บในรูปแบบ
CSV File แต่ต้องมีโครงสร้างของข้อมูลครบถ้วนตามที่โจทย์กําหนดให้ โดยนักศึกษาไม่จําเป็นต้องคํานึงถึงหลักความปลอดภัยใด ๆ ของระบบ แต่นักศึกษาจะต้องมีตัวอย่างข้อมูลวัวอย่างน้อย 10 ตัว สีละไม่น้อยกว่า 3 ตัว และอายุเป็นแบบสุ่มตั้งแต่ 0 - 10 ปี โดยมีจํานวนปีและเดือนด้วย
3) ระบบจะรับรหัสจากหน้าจอ GUI ให้นักศึกษาสร้างหน้าจอ GUI เพื่อรับรหัสอย่างง่าย โดยโปรแกรมจะตรวจสอบข้อมูลตามความเหมาะสมที่ควรจะเป็น เช่น ข้อมูลที่ใส่เข้ามาเป็นตัวเลขหรือไม่ ข้อมูลเป็นตัวเลข 8 ตัวจริงหรือไม่ เลขไม่นําหน้าด้วยเลข 0 ใช่หรือไม่ เป็นต้น หลังจากนั้นระบบจะแสดงข้อมูลของวัวจากรหัสที่รับเข้ามา หากไม่เจอรหัสจะต้องแจ้งข้อผิดพลาดให้เห็น หากเจอจะแสดงหน้าจอพร้อมปุ่มให้รีดนม โดยที่หากเป็นวัวสีขาวจะมีปุ่มเพิ่มให้ป้อนมะนาวเพื่อผลิตนมเปรี้ยว ดังนั้น ให้ทํา View สําหรับวัวทั้งสองแบบแยกกันให้ชัดเจน
4) เมื่อกดปุ่มรีดนม วัวแต่ละตัวจะให้นม 1 ขวดตามประเภทของวัว (วัวสีขาวให้นมจืด วัวสีขาวกินมะนาวให้นมเปรี้ยว วัวสีน้ําตาลให้นมช็อกโกแลต) แต่วัวสีขาวจะมีโอกาสร้อยละ 0.5 คูณด้วยอายุส่วนของเดือนที่เพี้ยนไปผลิตนมถั่วเหลือง ส่วนวัวสีน้ําตาลจะมีโอกาสร้อยละ 1 คูณด้วยอายุส่วนของปีที่เพี้ยนไปผลิตนมอัลมอนด์หากเกิดเหตุการณ์นี้ นมที่ออกมาจะใช้ไม่ได้ ต้องทิ้ง เราจะเรียกวัวเหล่านี้ว่า BSOD (Breast Supply Operation Down – วัวผลิตน้ํานมไม่ได้) และวัวเหล่านี้จะกลายเป็นสีฟ้า ใช้การไม่ได้อีกจนกว่าจะกดปุ่ม Reset แต่ถ้าวัวที่กินมะนาวแล้ว จะไม่เกิดเหตุการณ์นี้ขึ้น จําเป็นจะต้องแจ้งผู้ใช้ถึงเหตุการณ์นี้ด้วย ให้ทํา Model สําหรับคํานวณวัวทั้งสองแบบแยกกันให้ชัดเจน
5) หลังรีดนมแต่ละครั้ง จะมีข้อมูลรายงานแสดงให้เห็นว่าวัวแต่ละตัวผลิตนมไปแล้วกี่ขวด และนมทั้ง 3 ชนิดที่ ใช้ได้มีรวมกันแล้วกี่ขวด รวมทั้งมีปุ่มให้ Reset วัวที่เกิดอาการ BSOD ทั้งหมดด้วย ผู้ใช้จะกด Reset หรือไม่ ณ ตรงนี้ก็ได้ แต่ถ้าไม่ Reset ครั้งต่อไปที่ใส่รหัสของวัวตัวนั้นจะต้องแสดงข้อผิดพลาด เมื่อผู้ใช้ดูรายงานสรุปแล้ว ให้มีปุ่มกลับไปหน้าจอในข้อ 2 เพื่อรอรับรหัสวัวตัวต่อไป

เลือกทำข้อ2 
Model.java มีฟังก์ชั่นโหลดข้อมูลวัวจากไฟล์ CSV, รีเซ็ต BSOD ของวัวทุกตัว, แสดงรายงาน, ค้นหาวัวด้วยรหัส, นับนมที่ผลิตขึ้นมาแล้ว ซึ่งใน model มีคลาส Cow ซึ่งเก็บรหัส, สี, ปี, อายุ, isBSOD โดยรหัส, สี, ปี, อายุ มีเมธอด get และ isBSOD สามารถ set Boolean
Controller.java  มีฟังก์ชั่นค้นหาวัว ผลิตนม รีเซ็ต BSOD, ได้รับรายงาน, แสดง InputView, ได้รับ model
InputView.java เป็น GUI ให้กรอกรหัสของวัวถ้ามีข้อมูลรหัสจะแสดงหน้า MilkCowView ถ้าไม่ไม่มีข้อมูลรหัสจะขึ้นแจ้งเตือนว่ารหัสผิด
MilkCowView.java เป็น GUI ให้รืดนมวัว โดยถ้าเป็นวัวสีขาว จะแสดงให้รืดนมสองแบบคือ ให้นมจืด กับให้นมเปรี้ยว ถ้าเป็นสีน้ำตาลจะให้นมช็อกโกแลต โดยวัวสีขางมีโอกาศเกิด BSOD 0.05% * เดือน วัวสีน้ำตาล BSOD 0.1% * ปี ซึ่งวัวสีขาวที่กินมะนาวให้นมเปรี้ยวจะไม่เกิด ซึ่งไม่ได้เขียนแจ้งไว้ที่ GUI โดยหลังเกิด BSOD มีสองทางคือจะกด รีเซ็ต BSOD หรือ กดกลับไปกรอกรหัสวัวใหม่ แต่ถ้ากรอกรหัสวัวที่ BSOD อยู่แต่ไม่ได้รีเซ็ตจะกดรีดนมไม่ได้
